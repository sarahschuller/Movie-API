Movie API

A web application that provides users with access to information about a variety of movies, directors, and genres. Users are able to sign up, update their personal information, and create a list of their favorite movies.

Features

Homepage of Movie App

Request Type: GET
Request URL: /
Message: Welcome to my Horror Movie App!

Return a list of all movies to the user

Request type: GET
Request URL: /movies/:title
Returns an array of movies within the database as a JSON object

Return data about movies by title

Request type: GET
Request URL: /genres/:title
Returns a JSON object with data (description, genre, director, year released) about a single movie by title

Example response:
{ title: "...", year: "...", genre: "...", director: "...", description: "..." }

Return data about a director

Request type: GET
Request URL: /directors/:name
Returns a JSON object with information/data about the movie's director
Example response:
{ director:"..." birthday:"..." bio:"..." }

Register new users to the application

Request type: POST
Request URL: /users
Create a JSON object with data about a newly registered user
Example response:
{ name: "John Doe" username: "johndoe123" email: "johndoe@example.com" password: "abcd123" } Message Response: "New user has been registered"

Users are able to update their information

Request type: PUT
Request URL: /users/:username
Example response:
{ name: "John Doe" age: 30 username: "johndoe123" email: "johndoe@example.com" password: "efgh456" } Message Response: "User information has been updated"

Allow users to add a movie to their favorites list

Request type: POST
Request URL: /users/:username/movies/:movieId
Create a JSON object with data about a user's favorite movies
Message response:"Movie has been added to user favorites"

Allow users to remove movies from their favorites list

Request type: DELETE
Request URL: /users/:username/movies/:movieId
Removes an item from the JSON object with the user's list of favorite movies
Message response: "Movie has been removed from favorites"

All current users to delete their registration

Request type: DELETE
Request URL: "/users"
Removes a user account from the database
Message response: "Your account was successfully deleted"
